## 基本ルール

### ゲームの目的
50階層のダンジョンを踏破し、10体のボスを倒すこと。

### ゲーム開始時のステータス
- HP: 100
- 最大エネルギー: 7
- 手札枚数: 6枚/ターン
- デッキ: 10枚（攻撃5枚、防御4枚、スキル1枚）
- ゴールド: 50

## バトルシステム

### ターンの流れ
1. **ドローフェーズ**: 山札から6枚引く
2. **プレイヤーターン**: エネルギーを使ってカードをプレイ
3. **ターン終了**: 「ターン終了」ボタン or カード/エネルギー使い切りで自動終了
4. **敵ターン**: 敵が順番に行動（攻撃/防御/強化/弱体化）
5. 次のターンへ（1に戻る）

### エネルギー
- 毎ターン開始時に最大値（7）まで回復
- カードをプレイするとコスト分消費
- 未使用エネルギーは次のターンに持ち越し**しない**

### ブロック（防御値）
ブロックは**一時的な防御バリア**です。

**仕組み**:
1. 防御カードを使うと「ブロック」を獲得する
2. 敵から攻撃を受けると、まずブロックが減る
3. ブロックが0になってから、HPが減る
4. **ブロックは次のターン開始時に0にリセットされる**

**例**:
- あなたのブロック: 15
- 敵の攻撃: 20ダメージ
- 結果: ブロック15で吸収 → 残り5ダメージがHPに

**重要ポイント**:
- ブロックは**そのターン限り**有効（持ち越し不可）
- 敵の攻撃予告を見て、必要な分だけブロックを積む
- 過剰なブロックは無駄になる（次ターンでリセットされるため）

### ダメージ計算
```
最終ダメージ = (基本ダメージ + 闘志) × 躊躇補正 × 不安補正 - ブロック
```
1. 基本ダメージを計算
2. 闘志（strength）を加算
3. 躊躇（weak）がある場合、**75%に減少**
4. 不安（vulnerable）がある場合、**150%に増加**
5. ブロックで軽減

## ステータス効果（重要！）

### バフ（味方に有利な効果）

| アイコン | 名前 | 効果 | 例 |
|---------|------|------|-----|
| 💪 | **闘志** | 攻撃ダメージに+（スタック数）加算 | 闘志3 → 全攻撃+3ダメージ |
| 🏃 | **克己** | ブロック値に+（スタック数）加算 | 克己2 → 全ブロック+2 |
| ✨ | **調和** | ターン終了時にHP回復 | 調和3 → 毎ターン3HP回復 |

### デバフ（敵に有利な効果）

| アイコン | 名前 | 効果 | 計算例 |
|---------|------|------|--------|
| 💔 | **不安** | 受けるダメージが**1.5倍** | 10ダメージ → 15ダメージ |
| 😵 | **躊躇** | 与えるダメージが**0.75倍** | 10ダメージ → 7ダメージ |
| 🔻 | **倦怠** | 得るブロックが**0.75倍** | 10ブロック → 7ブロック |
| ☠️ | **苦悩** | ターン終了時にダメージ | 苦悩3 → 毎ターン3ダメージ |

### ステータスの持続
- スタック数は効果の強さ
- 持続ターン数があるものは、ターン終了時に1減少
- 0になると効果消滅

**例**: 「不安2（2ターン）」
- 2ターンの間、受けるダメージ1.5倍
- 2ターン後に効果消滅

## カードタイプ

### 攻撃カード（赤）🗡️
敵にダメージを与える。

| バリエーション | 効果 |
|--------------|------|
| 単体高威力 | 1体に大ダメージ |
| 全体攻撃 | 全ての敵にダメージ（威力低め） |
| 不安付与 | ダメージ + 敵に不安 |
| 躊躇付与 | ダメージ + 敵に躊躇 |
| 闘志獲得 | ダメージ + 自分に闘志バフ |

### 防御カード（青）🛡️
ブロックを得る。

| バリエーション | 効果 |
|--------------|------|
| 高ブロック | 大量のブロック |
| 克己付与 | ブロック + 克己バフ |
| ドロー付き | ブロック + カードを引く |
| 反撃 | ブロック + 少量ダメージ |
| 躊躇付与 | ブロック + 敵に躊躇 |

### スキルカード（緑）✨
多様な効果。

| バリエーション | 効果 |
|--------------|------|
| ドロー | カードを複数枚引く |
| エネルギー | エネルギーを獲得 |
| 闘志バフ | 闘志を大量獲得 |
| 全体デバフ | 全敵に不安+躊躇 |
| 複合 | ダメージ+ブロック+ドロー |

### 回復カード（緑）💚
HPを回復。

| コスト | 回復量 | 追加効果 |
|-------|--------|---------|
| 1 | 8 HP | なし or ブロック |
| 2 | 18 HP | なし or ブロック or 調和 |
| 3 | 30 HP | 調和バフ付き |

## リスク/リターンカード（特殊カード）

### 概要
通常のカードより**ハイリスク・ハイリターン**な特殊カードです。
うまく使えば戦局を一変させますが、リスクも伴います。

### HPコストカード（HP消費で強力効果）
HPを消費して強力な効果を得るカード。**HP100基準で18〜25%を消費**するハイリスク設計。

| カード名 | コスト | 効果 | レア度 |
|---------|--------|------|--------|
| 血の契約 | 2 | HP25消費 → 150ダメージ | ★★★★ |
| 命を燃やす | 3 | HP20消費 → 全体100ダメージ | ★★★★ |
| 捨身の守り | 2 | HP18消費 → 90ブロック | ★★★ |
| 血の代償 | 2 | HP22消費 → 闘志+6、克己+5 | ★★★★ |

**重要ルール**:
- **HPがHP消費量以下だと使用不可**（死亡を防ぐため）
- 例: HP20でHP25消費カードは使用不可

**ポイント**:
- HPに余裕がある時に使用
- 回復手段と併用すると効果的
- ボス戦の切り札として有効

### 条件付きカード（HP状態で効果変動）
プレイヤーの状態によって効果が変わるカード。

| カード名 | コスト | 通常効果 | 条件達成時 | 条件 |
|---------|--------|---------|-----------|------|
| 背水の陣 | 3 | 50ダメージ | 100ダメージ | HP50%以下 |
| 死中求活 | 2 | 30ブロック | 90ブロック | HP30%以下 |
| 余裕の一撃 | 2 | 40ダメージ | 80ダメージ | HP50%以上 |
| 窮鼠の反撃 | 4 | 全体40ダメージ | 全体100ダメージ | HP30%以下 |

**条件一覧**:
| 条件 | 説明 |
|------|------|
| HP50%以下 | HPが最大HPの半分以下 |
| HP50%以上 | HPが最大HPの半分以上 |
| HP30%以下 | HPが最大HPの30%以下（瀕死状態） |

**ポイント**:
- 条件を満たすと効果が**2〜3倍**に増加
- 「背水の陣」型：ピンチ時に真価を発揮
- 「余裕の一撃」型：HP管理が重要

## ストックカード（重要システム）

### ストックカードとは？
報酬で獲得したカードを**最大5枚までストック**しておける特別なスロットです。

### 使い方
1. **設定**: 報酬画面でカードを選択後、「📦 ストック」を選ぶ
2. **使用**: バトル中、ストックカードボタンをタップして使用
3. **制限**: 最大5枚まで保持可能

### 活用例
- **回復カード**をストック → HPが減った時に即使用
- **強力な攻撃カード**をストック → ボス戦で切り札として使用
- **バフカード**をストック → 戦闘開始直後に発動
- **コンボ用カード**を複数貯めておく → 組み合わせて一気に発動

### 注意点
- ストックカードは**デッキには入らない**（手札として引かれない）
- 使用すると消滅（1回限り）
- 戦闘ごとにリセット**されない**（次の戦闘でも使える）
- 5枚貯まると新しいカードはストックに追加できない

## コンボシステム（重要！）

### コンボとは？
同じターン内に**関連するカード**を使用すると、特別な**コンボ効果**が発動します。
哲学者同士の関係性（師弟、対立、同流派など）に基づいてコンボが決まります。

### コンボの種類

| コンボタイプ | 条件 | 例 |
|-------------|------|-----|
| **著者コンボ** | 同じ哲学者のカード2枚 | ニーチェ×2 → 「力への意志」 |
| **師弟コンボ** | 師弟関係の哲学者 | ソクラテス+プラトン → 「アテネの系譜」 |
| **対立コンボ** | 対立関係の哲学者 | ニーチェ+プラトン → 「価値の転覆」 |
| **流派コンボ** | 同じ流派の哲学者2人 | デカルト+スピノザ → 「理性の光」 |

### コンボ効果の例

| コンボ名 | 発動条件 | 効果 |
|---------|---------|------|
| 🔥 **力への意志** | ニーチェ×2 | 15ダメージ + 闘志+2(2T) |
| 📜 **アテネの系譜** | ソクラテス/プラトン/アリストテレス | 3ドロー + 克己+1 + 5回復 |
| ⚡ **弁証法的衝突** | 対立関係の2人 | 全体25ダメージ |
| 💡 **理性の光** | 合理主義者×2 | 2ドロー + 1エネルギー |
| 🦅 **ドイツ精神** | ドイツ観念論者×2 | 12ブロック + 克己+1 |

### コンボのコツ
1. **著者を確認**: カードの元になった哲学者を覚えておく
2. **ストックを活用**: コンボ用カードをストックに貯めておく
3. **流派を意識**: 実存主義、合理主義など流派でまとめると発動しやすい
4. **師弟関係**: ソクラテス→プラトン→アリストテレスなど有名な系譜を狙う

### 主な哲学者と流派

| 流派 | 哲学者 |
|------|--------|
| **古代ギリシャ** | ソクラテス、プラトン、アリストテレス |
| **合理主義** | デカルト、スピノザ、ライプニッツ |
| **ドイツ観念論** | カント、フィヒテ、ヘーゲル |
| **実存主義** | キルケゴール、ニーチェ、ハイデガー、サルトル、カミュ |
| **現象学** | フッサール、ハイデガー |
| **心理学** | フロイト、ユング |

## 敵の行動

敵ターンでは、各敵が順番に以下のいずれかの行動を取ります：

| 行動 | 効果 |
|------|------|
| **攻撃** | プレイヤーにダメージ（ブロックで軽減可能） |
| **防御** | 敵自身がブロックを得る |
| **自己強化** | 闘志アップ（以降の攻撃が強くなる） |
| **デバフ** | プレイヤーに躊躇デバフを付与 |

**ポイント**: 敵の行動は予告されないため、常に防御を意識しつつ攻めましょう。

## ゲームのコツ

1. **攻撃と防御のバランスを意識する**
   - 敵の攻撃に備えて毎ターンある程度の防御を確保
   - 複数の敵がいる場合は集中攻撃で各個撃破

2. **全体攻撃は複数の敵に有効**
   - 敵が3体の時、全体攻撃は実質3倍の価値
   - 威力は単体攻撃より低いがトータルダメージは高い

3. **デバフを活用**
   - 不安付与 → 次の攻撃が1.5倍に
   - 躊躇付与 → 敵の攻撃力を下げる

4. **回復は計画的に**
   - ボス撃破時: HP全回復
   - 回復カード: 戦闘中に使用
   - ストックに回復カードを保持しておくと便利

5. **エネルギーは使い切る**
   - 余ったエネルギーは次ターンに持ち越せない
   - 手札やエネルギーがなくなると自動でターン終了

6. **コンボを狙う**
   - 同じ哲学者のカードを組み合わせる
   - 師弟関係や対立関係を覚えておく
   - ストックにコンボ用カードを貯めておく
